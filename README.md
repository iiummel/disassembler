disassembler
============

Final Project for CSS 422
This project disassembles machine code into 68k assembly language.

<br>Data Registers
<br>D0 : Traps
<br>D1 : Traps
<br>D2 : I/O
<br>D3 : Share
<br>D4 : Share
<br>D5 : Share
<br>D6 : OP
<br>D7 : OP

<br>Address Registers
<br>A0 : Share 
<br>A1 : Share (OP/Jump Tables)
<br>A2 : Share
<br>A3 : Good Buffer
<br>A4 : Bad Buffer
<br>A5 : Start Address
<br>A6 : End Address
<br>
